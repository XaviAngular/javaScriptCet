Obtener valor de formulario

Para obtener el valor de un campo de formulario, jQuery dispone de la funcion .val(). Por ejemplo, si un campo de formulario tiene el id="campo", para obtener y guardar en variable el valor se usa var variable=$('#campo').val().

Introducir valores en campos de formulario

Para realizar la operación inversa, es decir, aplicar un valor a un campo de formulario, se usa la misma funcion .val() pero declarando el valor entre los paréntesis. Es la técnica que se usa para editar información. Por ejemplo, para añadir el valor "Macarrones" al campo de formulario con id="nombre" se usa $('#nombre').val('Macarrones')
